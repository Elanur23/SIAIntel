## 🚨 Troubleshooting

### Issue: Seal Not Applied
**Symptoms**: Clickbait terms still present in output

**Solution**:
1. Check `applyGoldenRuleDictionary()` is called before distribution
2. Verify `job.goldenRuleApplied = true` is set
3. Review console logs for conversion count
4. Test with `test-seals-validation.ts` to isolate issue

---

### Issue: Missing Verification Badge
**Symptoms**: Widget does not show "Verified by SIA Sentinel"

**Solution**:
1. Check `showAttribution: true` in widget config
2. Verify `generateBarometerEmbed()` includes badge HTML
3. Inspect widget CSS for `.sia-verification-badge` styles
4. Test widget generation API directly

---

### Issue: Global Links Not Injected
**Symptoms**: No "Global Intelligence Network" section in content

**Solution**:
1. Check `addGlobalContextLinks()` is called for each language
2. Verify `job.globalLinksInjected = true` is set
3. Review content length (must be >100 characters)
4. Test with sample content in validation script
