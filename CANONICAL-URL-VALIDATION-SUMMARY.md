### 2. Canonical URL Construction

## TROUBLESHOOTING

### Issue: Canonical Mismatch

**Symptoms:**
- Google-selected ≠ User-declared canonical

**Solutions:**
1. Request re-indexing in GSC
2. Check for redirect chains
3. Verify internal links use canonical format
4. Wait 1-2 weeks for re-crawl

### Issue: Not Indexed

**Symptoms:**
- "Discovered - currently not indexed"

**Solutions:**
1. Check robots.txt allows crawling
2. Verify no noindex meta tag
3. Improve content quality
4. Build internal links
5. Request indexing via GSC

### Issue: Duplicate Content

**Symptoms:**
- "Alternate page with proper canonical"

**Solutions:**
1. Ensure only canonical URLs in sitemap
2. Add 301 redirects from old URLs
3. Update internal links
4. Remove URL variations
