# Intelligence Depth Modal - Bloomberg Terminal Style ✅

## Status: COMPLETE

The Intelligence Report Modal has been successfully updated with Bloomberg Terminal aesthetic.

## What Was Done

### 1. Bloomberg Terminal Styling Applied
- **Sharp Corners**: Removed all rounded corners (`rounded-lg` → no rounding)
- **Pure Black Background**: Changed from gradient backgrounds to pure `bg-black`
- **Amber Border**: Updated border from gray to `border-[#FFB800]` (2px width)
- **Amber Accents**: All primary UI elements use amber (#FFB800) color
- **Monospace Typography**: Applied `font-mono` throughout for terminal aesthetic

### 2. ESC Key Handler
- Added `useEffect` hook to listen for ESC key press
- Pressing ESC closes the modal instantly
- Event listener properly cleaned up on unmount

### 3. Close Button Update
- Changed from icon-only (X) to text button
- New text: `CLOSE_TERMINAL [ESC]`
- Styled with amber background and black text
- Positioned in top-right corner of header

### 4. Percentage Bars for Metrics
- **Sentiment Bar**: Shows confidence percentage with color-coded bar
  - BULLISH: Green (#00FF00)
  - BEARISH: Red (#FF0000)
  - NEUTRAL: Amber (#FFB800)
- **Market Impact Bar**: Shows impact as percentage (0-100%)
  - High Impact (≥8): Red
  - Medium Impact (6-7): Amber
  - Low Impact (<6): Green
- Bars have dark gray background with colored fill
- Percentage values displayed below each bar

### 5. Content Sections Styling
- **Executive Summary**: Amber border, black background
- **Sovereign Insight**: Green border (#00FF00), black background
- **Risk Assessment**: Red border (#FF0000), black background
- All text uses monospace font for terminal feel
- Section headers have bottom border separator

### 6. Professional Typography
- Font size: 11-12px for body text
- Uppercase tracking for headers
- Tabular numbers for metrics
- Gray text (#gray-400) for readability on black

**Props Interface**:
```typescript
interface SIAReportModalProps {

  report: {

    region: string

  }
```

## User Interaction

### Opening Modal
- Click any intelligence row in center panel
- Modal appears with backdrop blur
- Displays full Gemini report with all fields

### Closing Modal
1. Click "CLOSE_TERMINAL [ESC]" button (top-right)
2. Press ESC key on keyboard
3. Click backdrop (outside modal)

## Integration

The modal is already integrated in `app/page.tsx`:

```typescript
const [selectedReport, setSelectedReport] = useState<IntelligenceItem | null>(null)

// Click handler on intelligence rows
onClick={() => setSelectedReport(item)}

// Modal component
<SIAReportModal
  isOpen={!!selectedReport}
  onClose={() => setSelectedReport(null)}
  report={selectedReport}
/>
```
