## Troubleshooting

### Common Issues

**Issue**: Sources not displaying  
**Solution**: Check `sources` array is not empty and has valid source types

**Issue**: AI disclosure not showing  
**Solution**: Verify `aiDisclosure` object has all required fields

**Issue**: Correction notice missing  
**Solution**: Ensure `correctionPolicy` string is provided

**Issue**: Wrong language labels  
**Solution**: Check `language` field matches supported languages (en, tr, de, es, fr, ar)
