### 2. AdSense Application Process
1. **Sign up**: Visit [google.com/adsense](https://www.google.com/adsense)
2. **Submit site**: Enter your website URL
3. **Add AdSense code**: Place code in `<head>` section
4. **Wait for review**: Typically 1-2 weeks
5. **Address feedback**: If rejected, review and fix issues

### 3. AdSense Code Placement
**Recommended locations**:
- Between article paragraphs (native ads)
- Sidebar (display ads)
- Below article content (related content ads)
- Header/footer (banner ads)

**Implementation**:
```typescript
// components/AdBanner.tsx
export function AdBanner({ slot }: { slot: string }) {
  return (
    <ins
      className="adsbygoogle"
      style={{ display: 'block' }}
      data-ad-client="ca-pub-XXXXXXXXXX"
      data-ad-slot={slot}
      data-ad-format="auto"
      data-full-width-responsive="true"
    />
  )
}
```

```bash
# AdSense Slots
```

```typescript

  async
```
