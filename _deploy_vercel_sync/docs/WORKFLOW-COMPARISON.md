## 🚀 Başlangıç

### Hızlı Kurulum
```typescript
import { aiWorkflowAutomation } from '@/lib/ai-workflow-automation'

// Hazır şablon kullan
const workflow = await aiWorkflowAutomation.createNewsPublishingWorkflow()

// Veya custom workflow oluştur
const customWorkflow = await aiWorkflowAutomation.createWorkflow({
  name: 'My Custom Workflow',
  trigger: { type: 'schedule', config: { schedule: '0 */2 * * *' } },
  actions: [
    { id: 'action1', type: 'fetch_rss', name: 'Fetch RSS', config: {} },
    { id: 'action2', type: 'ai_generate', name: 'Generate Content', config: {} }
  ],
  errorHandling: { onError: 'retry', maxRetries: 3, retryDelay: 5000 }
})

// Workflow'u çalıştır
await aiWorkflowAutomation.executeWorkflow(workflow.id)
```
