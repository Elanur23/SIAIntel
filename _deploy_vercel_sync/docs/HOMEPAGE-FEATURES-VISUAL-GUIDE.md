## 🎭 Animation Effects

### Hover Effects
```
Intelligence Card:
  Normal:  border: #C0C0C0/20
  Hover:   border: #FFD700/50 + scale(1.03) + gold glow

Language Flag:
  Normal:  opacity: 0.6
  Hover:   opacity: 1.0 + scale(1.3)
  Active:  ring-2 ring-[#FFD700]

Market Ticker:
  Continuous horizontal scroll (30s duration)
```

### Pulse Effects
```
LIVE Indicator:
  animate: scale [1, 1.2, 1] + opacity [1, 0.5, 1]
  duration: 2s
  repeat: Infinity

Active Nodes:
  Green dot pulses when active
  Gray dot static when inactive
```

### Transitions
```
Page Load:
  Header:  y: -100 → 0 (0.5s)
  Hero:    opacity: 0 → 1, x: ±50 → 0 (0.6s)
  Cards:   opacity: 0 → 1, y: 50 → 0 (staggered)

Language Switch:
  Intelligence cards: fade out → filter → fade in (0.3s)
  Video: crossfade to new language (0.5s)
```
