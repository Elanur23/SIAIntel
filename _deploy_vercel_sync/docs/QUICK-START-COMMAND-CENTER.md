# Quick Start: Command Center

## 🚀 5-Minute Setup

### 1. Access Command Center

Navigate to: `/admin/command-center`

Or from admin dashboard: Click "Command Center" card

### 2. Paste Intelligence

```
Example Input:
Federal Reserve signals potential rate cuts in Q2 2026 as inflation 
shows sustained decline to 2.3%. Core PCE at 2.3%, down from 2.8%.
Market pricing in 75bps of cuts by year-end. Tech sector rallies 3.2%.
```

### 3. Generate Content

Click: **"GENERATE GLOBAL INTELLIGENCE (6 REGIONS)"**

Wait ~60 seconds for:
- ✅ DIP Analysis (10-15s)
- ✅ SEO Metadata (2-3s)
- ✅ Global Content (30-45s)

### 4. Review Output

You'll see 6 language cards:
- 🇦🇪 Arabic ($440 CPM) - PREMIUM
- 🇺🇸 English ($220 CPM)
- 🇫🇷 Français ($190 CPM)
- 🇩🇪 Deutsch ($180 CPM)
- 🇪🇸 Español ($170 CPM)
- 🇹🇷 Türkçe ($150 CPM)

**Total**: $1,350 CPM potential

### 5. Deploy

Click: **"🚀 DEPLOY TO ALL NETWORKS (LIVE)"**

Content goes live immediately on homepage!

## 🔧 Troubleshooting

**Nothing happens when I click generate?**
- Check you pasted content in textarea
- Verify GEMINI_API_KEY is set in .env

**Processing stuck?**
- Wait up to 60 seconds
- Check browser console for errors
- Refresh page and try again

**Deploy button not working?**
- Ensure all processing completed
- Check both DIP and SEO generated
