# SIA (STRATEGIC INTELLIGENCE AGENCY) - LINGUISTIC & SEMANTIC AUDIT (LISA) PROTOCOL v1.0

This protocol defines the final automated quality gate for all SIA Intelligence Reports. No report shall be published if it fails the LISA Audit.

## 1. SEMANTIC PURITY (AI-ISM ELIMINATION)
Detect and remove non-substantive phrases commonly generated by LLMs.
- **FORBIDDEN:** "In the ever-evolving world of...", "It is important to remember...", "A deep dive into...", "Navigate the landscape...".
- **RULE:** If a sentence contains zero data points or zero technical terms, it is marked as **NOISE** and must be rewritten.

## 2. INSTITUTIONAL TERMINOLOGY PARITY
Every language node must use the professional financial equivalent, not common or slang terms.
- **TR Node:** "Fiyat patlaması" -> **"Fiyat Rallisi"**, "Para çokluğu" -> **"Likidite Bolluğu"**.
- **EN Node:** "Price jump" -> **"Valuation Surge"**, "Big players" -> **"Institutional Entities"**.
- **DE Node:** "Geldfluss" -> **"Kapitalfluss"**, "Marktbewegung" -> **"Marktdynamik"**.
- **RULE:** Use the **Oxford/Reuters Financial Dictionary** standard for each local node.

## 3. SYNTACTIC FLOW & DENSITY
- **SENTENCE_LENGTH:** Maximum 25 words per sentence. Complex thoughts must be broken into data-dense clusters.
- **INFORMATION_DENSITY:** Every paragraph must contain at least one **GOLDEN_KEYWORD** or one **TECHNICAL_METRIC**.
- **RULE:** Low-density paragraphs are flagged as **SUB-STANDARD**.

## 4. LOCALIZED DIALECT (ŞİVE) & SLANG DETECTION
The tone must be **GLOBAL_COLD_LOGIC**.
- **ZERO_TOLERANCE:** For regional slang, "shive", or emotional adjectives (e.g., "amazing", "incredible", "korkunç", "harika").
- **RULE:** Emotional bias detected above 5% on the sentiment-to-logic ratio will trigger a **REJECT** signal.

## 5. MOJIBAKE & ENCODING INTEGRITY
- Automated check for broken characters (e.g., Ã¼, Ä°, Ğ\u201C).
- Ensure UTF-8 consistency across all 9 nodes (EN, TR, DE, FR, ES, RU, AR, JP, ZH).

## 6. FINAL GO/NO-GO CRITERIA
| Metric | Threshold | Action on Failure |
|--------|-----------|-------------------|
| Semantic Noise | < 3% | Automated Removal |
| Terminal Accuracy | 100% | Hard Reject |
| Regional Slang | 0% | Hard Reject |
| Data Density | > 40% | Expansion Required |
